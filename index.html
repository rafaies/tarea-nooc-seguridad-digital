Semáforo de Riesgos:

Seguridad móvil, servicios en la nube y redes públicas

NOOC INTEF

Prácticas "seguras"

  • 1. Uso de funda o carcasa protectora del dispositivo seguridad física
  • 2. Proteger la tarjeta SIM con un código PIN bien elegido seguridad de acceso
  • 3. Apuntar, en lugar distinto al móvil, el número IMEI del dispositivo seguridad ante pérdidas o robos
  • 4. Instalar antivirus y mantenerlo actualizado seguridad del software
  • 5. Comprobar los permisos solicitados por una aplicación antes de descargar e instalar seguridad para su privacidad
  • 6. Por defecto tener las conexiones inalámbricas desactivadas seguridad en conexiones

Prácticas "con precaución"

  • 1. Uso de fundas o carcasas fantasía seguridad física
  • 2. Proteger acceso al dispositivo mediante Patrón de desbloqueo seguridad de acceso
  • 3. Instalar aplicación antirrobo para localizar el dispositivo y bloquearlo seguridad ante pérdidas o robos
  • 4. Instalar aplicación que monitorice el estado del dispositivo como CONAN Mobile seguridad del software
  • 5. Instalar aplicaciones leyendo sus cláusulas y condiciones seguridad para su privacidad
  • 6. Realizar copias de seguridad en servicios de almacenamiento en la nube seguridad en conexiones

Prácticas "peligrosas"

  • 1. Llevar el dispositivo en el bolsillo sin funda o carcasa seguridad física
  • 2. Usar como código PIN su año de nacimiento seguridad de acceso
  • 3. Dejar el dispositivo sobre la mesa en terrazas de verano seguridad ante pérdidas o robos
  • 4. "Permitir orígenes desconocidos" en los Ajustes de Seguridad del dispositivo para que se instalen aplicaciones de tiendas no oficiales seguridad del software
  • 5. Aceptar sin mirar ni leer detenidamente al descargar e instalar Apps seguridad para su privacidad
  • 6. Utilizar siempre "wifi públicas" para no gastar tarifa de datos seguridad en conexiones
